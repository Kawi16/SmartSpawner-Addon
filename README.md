## Requirements

- **Minecraft Version:** 1.21 - 1.21.8
- **Server Software:** Paper or compatible forks
- **Java Version:** 21+

## Building

```bash
git clone https://github.com/Kawi16/SmartSpawner-Addon.git
cd SmartSpawner-Addon
./gradlew build
```

The compiled JAR will be available in `build/libs/`
